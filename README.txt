Running:

Use python play.py PLAYER1 PLAYER2 to play. Default depth for MiniMax and AlphaBeta is 3.
    - human: Human Player
    - random: Random Player
    - minimax: MiniMax Player
    - alphabeta: AlphaBeta Player

To change the depth, you can either change the global constant for DEPTH_1 and DEPTH_2 in play.py or 
change the arguments passed to main().

Evaluation:

Use evaluation() in play.py to evaluation for mutiple times.
